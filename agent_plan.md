# 에이전트 서브그래프
최초 생성 시 노드 생성 위치가 이상함. 유저가 바로 볼 수 있는 위치였으면 함.


# 참조 스킬
기존 Declarative 스킬은 전역 배경지식임. 노드 그래프에 배치 불가능임. 이보다 조금 작은 단위로 사용할 '재사용 가능한 참고용 노드'가 필요함.
- 기존 노드( 에이전트 로컬 노드 포함. 트랜스퍼 스킬 제외.)의 하단에 새로운 포트를 생성, 즉 상/하 방향 연결임.
- 참조 스킬은 위에 포트가 있음. 프로시주얼 스킬/에이전트 아래에 참조 스킬들이 달리는 형태.
- 참조 스킬은 전역 정의지만 에이전트 로컬에서도 사용 가능함.
- 에이전트 로컬 레지스트리에 참조 스킬 영역 추가 필요함.
